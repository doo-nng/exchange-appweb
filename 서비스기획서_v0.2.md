## 01. 프로젝트 개요

### 서비스 한 줄 소개

> 달러 · 엔화 · 위안화 환율 트렌드를 한눈에 보고, 지금이 환전 적기인지 바로 판단할 수 있는 나만의 대시보드

### 왜 만드는가

비즈니스를 하다 보면 환전이 필요한 순간이 자주 생긴다. 그런데 그 순간마다 바로 환전하면 환율이 높을 때 손해를 볼 수 있다. 환율이 낮을 때 미리 환전해두고 싶은데, 지금 환율이 싼 건지 비싼 건지 직관적으로 알 수 있는 도구가 없다.

네이버나 은행 앱은 현재 환율만 보여주고, 트렌드를 한눈에 파악하기 어렵다. 내가 원하는 통화들만 모아서, 트렌드와 함께 "지금 환전하면 유리한지"를 바로 판단할 수 있는 나만의 도구를 만들기로 했다.

### 목표

| 구분 | 목표 |
|------|------|
| 단기 | 달러 / 엔화 / 위안화 환율을 최근 1년 트렌드와 함께 보여준다 |
| 장기 | 트렌드 분석으로 환전 적기를 자동으로 판단하고, 목표 환율 도달 시 알림을 준다 |

### 사용자

나 혼자 사용하는 개인 도구다. 맥북 브라우저와 아이폰에서 모두 쓸 수 있어야 한다. 아이폰에서는 홈 화면에 앱처럼 설치해서 네이티브 앱처럼 빠르게 열 수 있어야 한다.

## 02. 핵심 기능 정의

### MUST HAVE — 반드시 있어야 하는 기능

**환율 현황 확인**
- 달러(USD) · 엔화(JPY) · 위안화(CNY) 환율을 카드 형태로 한눈에 본다
- 매매기준율 / 살 때 / 팔 때를 구분해서 표시한다 (실제 환전 시 적용 금액 기준)
- 은행 공식 고시 환율 기준임을 항상 명시한다
- 마지막 업데이트 시각을 항상 표시한다 (오전 11시 이후 당일 기준, 이전이면 전일 기준임을 안내)
- 앱을 열면 자동으로 최신 데이터를 불러오고, 수동 새로고침도 가능하다

**트렌드 그래프**
- 최근 1년 환율 변화를 라인 그래프로 보여준다
- 기간 탭(1주 / 1개월 / 3개월 / 6개월 / 1년)으로 원하는 구간을 선택할 수 있다

**환전 적기 판단 도우미**
- 52주(1년) 최저가·최고가 범위에서 현재 환율이 어디쯤 있는지 Progress bar로 시각화한다
- "현재 달러는 1년 평균보다 X원 비싼 구간이에요" 같은 한 줄 문장으로 즉시 판단 근거를 준다
- Progress bar 색상: 평균 이하(저렴)면 파란색, 평균 이상(비쌈)이면 빨간색

**반응형 디자인**
- 맥북 브라우저와 아이폰 화면 모두에서 불편함 없이 사용할 수 있어야 한다

---

### NICE TO HAVE — 있으면 좋은 기능

- **목표 환율 알림** — 내가 원하는 환율에 도달하면 알림을 받는다
- **금액 계산기** — 원화 ↔ 외화를 즉시 환산한다 (예: 100만원이면 몇 달러인지)
- **AI 환율 인사이트** — 현재 환율 수준에 대한 자동 분석 코멘트를 제공한다
- **과거 데이터 확장** — 1년에서 최대 5년까지 조회 범위를 늘린다

---

### OUT OF SCOPE — 이번 버전에서는 하지 않는 것

| 항목 | 제외 이유 |
|------|-----------|
| 뉴스 피드 | 별도 서비스로 분리 예정 |
| 회원가입 / 로그인 | 개인 도구이므로 불필요 |

## 03. 화면 구성 (UX)

**디자인 방향:** 토스 · 스위치온 스타일 — 화이트 베이스, 카드형 컴포넌트, 넉넉한 여백. 복잡하지 않고 빠르게 파악되는 화면.

### 화면 흐름

앱을 열면 대시보드 하나에서 모든 걸 볼 수 있어야 한다. 별도 탭 이동 없이 스크롤로 이어지는 구조.

```
┌─────────────────────────────────┐
│  환율 카드 (USD / JPY / CNY)     │  ← 현재 환율 + 전일 대비 변화
│  마지막 업데이트 시각             │
│  새로고침 버튼                   │
├─────────────────────────────────┤
│  환전 적기 판단 바                │  ← 통화별 52주 레인지 + 한줄 코멘트
├─────────────────────────────────┤
│  환율 그래프                     │  ← 기간 탭 + 라인 차트
└─────────────────────────────────┘
```

### 환율 카드

각 통화별로 카드 1장. 카드 안에는:
- 통화명 + 국가 표시
- 매매기준율 (크게)
- 살 때 / 팔 때 (작게)
- 전일 대비 변화 (상승/하락 색상 구분)

> **JPY 표시 주의:** 엔화는 1엔이 아닌 100엔 기준으로 고시되므로 카드에 "100엔 기준"을 명확히 표시한다.

### 환전 적기 판단 UI

```
USD · 달러 환전 적기 분석

52주 최저 1,280  [══════════════════●───]  52주 최고 1,430
                               ↑ 현재 1,381

현재 달러는 최근 1년 평균(1,340원)보다 41원 비싼 구간이에요.
조금 더 기다리는 걸 고려해볼 수 있어요.
```

- Progress bar 위치 = (현재가 - 52주 최저) / (52주 최고 - 52주 최저)
- 평균보다 싸면 파란색 + "지금 환전하기 좋은 구간이에요"
- 평균보다 비싸면 빨간색 + "조금 더 기다려보는 걸 고려해볼 수 있어요"

### 환율 그래프

- 통화 선택: USD · JPY · CNY (탭으로 전환)
- 기간 선택: 1W · 1M · 3M · 6M · 1Y
- 라인 차트. 심플하게, 불필요한 꾸밈 없이.

### 데이터 기준 안내

두 가지 데이터 출처를 사용하기 때문에, 현재 환율 카드와 과거 그래프의 수치가 소폭 다를 수 있다. 각 영역에 출처를 명시해 투명하게 처리한다.

- 현재 환율 카드 → 한국수출입은행 고시 기준
- 과거 그래프 → ECB(유럽중앙은행) 기준

## 04. 정책 & 제약

### 데이터 업데이트 정책

| 항목 | 정책 |
|------|------|
| 자동 갱신 | 앱을 열 때 자동으로 최신 데이터를 불러온다 |
| 수동 갱신 | 새로고침 버튼으로 즉시 재호출 |
| 업데이트 기준 | 영업일 오전 11시 이후 당일 데이터. 그 이전에 열면 전일 기준임을 안내 |
| 조회 범위 | 1년으로 시작, 안정성 검증 후 5년까지 확장 |

### 서비스 제약사항 (사용자 관점)

| 항목 | 내용 | 사용자 안내 방법 |
|------|------|-----------------|
| 오전 11시 이전 | 당일 환율이 아직 고시되지 않아 전일 데이터를 표시 | "오늘 환율 준비 중 · 전일 기준으로 표시" 안내 문구 노출 |
| JPY 단위 | 엔화는 100엔 기준 고시 (1엔 단위 아님) | 카드에 "100엔 기준" 명시 |
| 현재가 vs 그래프 수치 차이 | 데이터 출처 기준이 달라 수치가 소폭 다를 수 있음 | 각 영역에 출처 명시 |
| 아이폰 알림 | iOS 16.4 이상에서만 PWA 푸시 알림 지원 | 알림 기능은 Nice to Have 단계에서 별도 검토 |
