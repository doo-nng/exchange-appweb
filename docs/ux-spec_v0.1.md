## 3. 전체 레이아웃

### 페이지 구조 (단일 스크롤 페이지)

```
┌─────────────────────────────────────┐
│  [StaleDataBanner] (조건부)          │  ← 전일 기준 or 오프라인 시만 표시
├─────────────────────────────────────┤
│  [UpdateHeader]                     │  ← "수출입은행 기준 · MM월 DD일 HH:MM"
│                          [새로고침]  │
├─────────────────────────────────────┤
│  [CurrencyCard USD]                 │  ← + RangeBar 포함
│  [CurrencyCard JPY]                 │
│  [CurrencyCard CNY]                 │
├─────────────────────────────────────┤
│  [PeriodTab] 1W 1M 3M 6M 1Y        │
│  [RateChart]                        │
└─────────────────────────────────────┘
│  [Toast] (조건부)                    │  ← 에러 발생 시 하단 오버레이
```

### 반응형 레이아웃

**모바일 (< 768px):** 카드 1열 세로 스택. 카드 너비 100%.

**데스크탑 (≥ 768px):** 카드 3열 그리드. 그래프 전체 너비.

## 4. 화면별 컴포넌트 명세

### 4.1 UpdateHeader

**목적:** 데이터 신선도를 항상 표시하고 수동 새로고침을 제공한다.

| 상태 | 표시 텍스트 | 새로고침 버튼 |
|------|------------|--------------|
| 정상 (당일 기준) | `수출입은행 기준 · MM월 DD일 HH:MM 업데이트` | 활성 |
| 정상 (전일 기준) | `수출입은행 기준 · MM월 DD일 HH:MM 업데이트 (전일 기준)` | 활성 |
| 로딩 중 | 이전 텍스트 유지 | 스피너 (비활성) |
| 에러 | 이전 텍스트 유지 | 활성 (재시도 허용) |
| 데이터 없음 | `업데이트 정보 없음` | 활성 |

**새로고침 버튼 인터랙션:**
- 탭 → 버튼 내 순환 아이콘이 스피너로 전환 → 요청 완료 후 원래 아이콘으로 복귀
- 탭 피드백: 버튼 배경 `color-bg-base`로 순간 변화 (100ms)

---

### 4.2 StaleDataBanner

**목적:** 현재 표시 중인 데이터가 최신이 아님을 명확하게 알린다.

| 조건 | 배너 텍스트 | 색상 |
|------|------------|------|
| 오전 11시 이전 | `오늘 환율은 오전 11시 이후 업데이트됩니다 · 전일 기준으로 표시 중` | 노란색 배경 (`#FEF9C3`), 갈색 텍스트 |
| 오프라인 | `오프라인 상태 · 마지막 업데이트 [MM월 DD일 HH:MM]` | 회색 배경 (`#F3F4F6`), 회색 텍스트 |
| 데이터 없음 | 표시 안 함 (이 배너 아님, Toast 사용) | — |

배너는 닫기 버튼 없음. 조건이 해소되면 자동으로 사라짐.

### 4.3 CurrencyCard

**목적:** 통화 1개의 현재 환율 전체 정보를 카드 1장에 담는다.

#### 레이아웃 (내부 구조)

```
┌─────────────────────────────────┐
│ [국기 아이콘] USD  미국 달러    │  ← 상단: 통화 식별
│                  ↑ 8.50 (+0.6%) │  ← 우상단: 전일 대비
├─────────────────────────────────┤
│         1,381.00 원             │  ← 중앙 크게: 매매기준율
├─────────────────────────────────┤
│ 살 때   1,401.23    팔 때   1,360.77 │  ← 하단: 살 때/팔 때
├─────────────────────────────────┤
│ [RangeBar]                      │  ← 52주 레인지
│ 현재 달러는 1년 평균보다 41원 비싼... │
├─────────────────────────────────┤
│ 수출입은행 기준                  │  ← 출처 (아주 작게)
└─────────────────────────────────┘
```

#### 컴포넌트 상태별 표시

| 상태 | 매매기준율 | 살 때/팔 때 | 전일 대비 | RangeBar |
|------|-----------|------------|----------|----------|
| 정상 | 숫자 | 숫자 | 숫자 + 색상 화살표 | 표시 |
| 로딩 | SkeletonCard | — | — | — |
| null 값 | `--` | `--` | 숨김 | 숨김 |
| 에러 | `--` | `--` | 숨김 | "레인지 정보 없음" |

#### JPY 카드 특이사항

- 통화명 옆에 `100엔 기준` 뱃지(chip) 표시: 배경 `#EFF6FF`, 텍스트 `#2563EB`
- 매매기준율 아래: `(1엔 = [1엔값]원)` 소형 참고 텍스트 (선택 사항, open question)

#### 전일 대비 표시 규칙

| 변화 방향 | 텍스트 | 아이콘 | 색상 |
|----------|--------|--------|------|
| 상승 | `+[값] (+[%])` | ↑ | `color-signal-rise` |
| 하락 | `-[값] (-[%])` | ↓ | `color-signal-fall` |
| 변화 없음 | `0.00 (0.00%)` | — | `color-signal-neutral` |
| 계산 불가 | 숨김 | — | — |

### 4.4 RangeBar

**목적:** 현재 환율의 1년 내 위치를 시각적·언어적으로 전달한다.

#### 레이아웃

```
52주 최저  [══════════════●────]  52주 최고
1,280               현재           1,430
                   1,381

"현재 달러는 최근 1년 평균(1,340원)보다 41원 비싼 구간이에요."
```

#### Progress bar 규격

- 높이: 6px
- Track 색상: `#E5E7EB`
- Fill 색상: 저렴 구간 → `color-range-cheap`, 비싼 구간 → `color-range-expensive`
- 현재가 마커(●): 직경 12px 원, 흰 테두리 2px, Fill 색상과 동일
- 마커 위에 현재 환율 숫자 툴팁 (탭/호버 시)

#### 코멘트 문장 생성 규칙

```
평균 대비 차이 = 현재가 - 1년 평균가

차이 > 0 (비싸면):
  "[통화명]은 최근 1년 평균([평균값]원)보다 [차이]원 비싼 구간이에요."

차이 < 0 (저렴하면):
  "[통화명]은 최근 1년 평균([평균값]원)보다 [|차이|]원 저렴한 구간이에요."

차이 = 0:
  "[통화명]은 현재 최근 1년 평균과 동일한 수준이에요."
```

#### 상태별 처리

| 상태 | 표시 |
|------|------|
| 로딩 중 | Skeleton bar (회색 pulse 애니메이션) |
| 정상 | 계산값으로 표시 |
| 과거 데이터 없음 | `"레인지 정보를 불러오지 못했습니다"` 텍스트, bar 미표시 |

### 4.5 RateChart

**목적:** 선택 기간의 환율 변화 추이를 라인 차트로 보여준다.

#### 레이아웃

```
┌─────────────────────────────────────┐
│  [PeriodTab: 1W 1M 3M 6M 1Y]       │
│                                     │
│  [라인 차트 영역]                    │
│                                     │
│  ECB(Frankfurter) 기준              │  ← 출처 (우하단, 아주 작게)
└─────────────────────────────────────┘
```

#### 차트 스펙

- 차트 타입: 라인 차트 (Area 선택 가능)
- X축: 날짜. 탭 구간에 맞게 레이블 자동 조정 (1W: 일별, 1M: 주별, 1Y: 월별)
- Y축: KRW 환율. JPY: "100엔 당" 표기.
- 그리드라인: 수평 점선, 연한 회색
- 라인 색상: USD `#2563EB`, JPY `#D97706`, CNY `#DC2626`
- 데이터 포인트 탭/호버 시 툴팁: `[날짜] [통화]: [환율]원`

#### PeriodTab 인터랙션

- 탭 선택 → 선택된 탭 배경 `color-accent` + 텍스트 흰색, 비선택은 `color-text-secondary`
- 탭 전환 시 차트 데이터 즉시 업데이트 (fade 200ms)
- 기본 선택: `1M`

#### 차트 상태별 처리

| 상태 | 표시 |
|------|------|
| 로딩 중 | 차트 영역에 Skeleton (줄무늬 pulse) |
| 정상 | 라인 차트 |
| 선택 기간 내 데이터 없음 | `"해당 기간 데이터가 없습니다."` 중앙 텍스트 |
| 에러 | `"그래프를 불러오지 못했습니다. 새로고침 해주세요."` + 새로고침 링크 |

---

### 4.6 Toast (에러 알림)

**목적:** 일시적 에러 상황을 방해 없이 알린다.

- 위치: 화면 하단 중앙, z-index 최상위
- 표시 시간: 4초 후 자동 사라짐
- 탭으로 수동 닫기 가능
- 아이콘: 경고 삼각형(⚠) + 텍스트
- 배경: `#1A1A1A`, 텍스트: 흰색, border-radius: 12px

## 5. 교차 기능 인터랙션 패턴

### 로딩 상태

- 앱 최초 진입: 카드 자리에 SkeletonCard 표시 (실제 카드 높이와 동일), 그래프 영역에 Skeleton bar
- 새로고침 버튼 탭: 카드 교체 없이 값만 갱신 (카드 사라짐 없음)
- 전환 애니메이션: 새 값으로 바뀔 때 fade-in 150ms

### 에러 토스트 발생 조건

- 수출입은행 API 실패 시 1회만 토스트 (반복 실패 시 추가 토스트 없음)
- Frankfurter API 실패 시 토스트 + 인라인 에러 표시 (양쪽)

## 7. 접근성

| 요소 | 처리 |
|------|------|
| 새로고침 버튼 | `aria-label="환율 새로고침"` |
| 환율 상승/하락 아이콘 | `aria-hidden="true"` + 숫자 텍스트로 의미 전달 |
| Progress bar | `role="progressbar"`, `aria-valuenow`, `aria-valuemin`, `aria-valuemax` |
| PeriodTab | `role="tablist"`, 각 탭 `role="tab"`, `aria-selected` |
| SkeletonCard | `aria-busy="true"` |
| Toast | `role="alert"`, `aria-live="polite"` |
| 색상만으로 구분되는 정보 | 아이콘 + 색상 병용 (색맹 대응) |
