# 환율 트렌드 대시보드 — 프로젝트 컨텍스트

## 프로젝트 개요

달러·엔화·위안화 환율 트렌드를 한눈에 보고, 지금이 환전 적기인지 바로 판단할 수 있는 개인용 대시보드 웹앱 (PWA).

- 사용자: 나 혼자 (맥북 브라우저 + 아이폰 PWA)
- 목적: 비즈니스 환전 타이밍 판단 도구
- 기준 문서: `서비스기획서_v0.2.md`

## 문서 구조

```
exchange_appweb/
├── 서비스기획서_v0.2.md     # 기획 의도·기능·UX 방향 (최종본)
├── docs/
│   ├── prd_v0.1.md          # 기능 요구사항·수용기준·에러 상태 명세
│   ├── ux-spec_v0.1.md      # 화면별 컴포넌트 상태·인터랙션 명세
│   ├── api-data-spec_v0.1.md # 데이터 소스·모델·업데이트 정책
│   └── agents/
│       └── prd-writer-prompt.md  # PRD 작성 에이전트 지침
└── CLAUDE.md                # 이 파일
```

## 사용 스킬

이 프로젝트의 문서 작업에 사용하는 스킬:

| 스킬 | 목적 |
|------|------|
| `writing-prd` | 서비스기획서 → PRD 변환 |
| `writing-ux-spec` | 기획서 UX 방향 → 화면별 인터랙션 명세 |

## 핵심 제약사항 (개발 시 반드시 숙지)

| 항목 | 내용 |
|------|------|
| JPY 단위 | 수출입은행 API는 100엔 기준 고시. 1엔 단위로 오해하지 말 것 |
| 데이터 소스 이원화 | 현재 환율: 수출입은행 기준 / 과거 그래프: ECB(Frankfurter) 기준 — 수치 차이 있음, 각 출처 명시 필수 |
| 업데이트 기준 | 영업일 오전 11시 이후 당일 기준. 이전이면 전일 데이터 표시 + 안내 문구 필수 |
| "실시간 환율" 금지 | 은행 고시 환율은 일 1회 업데이트. 이 표현 사용 금지 |
| PWA 알림 제한 | iOS 16.4 이상만 PWA 푸시 알림 지원. 알림은 Nice to Have |

## 기능 우선순위

**P0 (MVP 필수)**
- 환율 카드 (USD·JPY·CNY) — 매매기준율/살 때/팔 때
- 52주 레인지 Progress bar + 1년 평균 대비 문장
- 1년 환율 트렌드 그래프 (기간 탭 포함)
- 마지막 업데이트 시각 표시 + 새로고침

**P1 (2차 작업)**
- 금액 계산기
- 환율 알림

**P2 (추후)**
- AI 인사이트
- 과거 데이터 5년 확장

## 개발 방향

- 가볍고 빠르게. 불필요한 의존성 최소화.
- 단일 HTML 파일 기반 (설치 없음, 배포 최단 경로)
- 토스·스위치온 스타일: 화이트 베이스, 카드형 컴포넌트, 넉넉한 여백
